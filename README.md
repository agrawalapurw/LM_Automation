# LM Automation - Email Extractor

Automated tool for extracting, validating, and processing Pre-MQL leads from Outlook emails.

## What It Does

- Extracts Pre-MQL emails from Outlook
- Validates leads against academic domains, excluded lists, and blacklisted countries
- Generates Excel files with Validation and Review sheets
- Automates web form submissions
- Moves emails to appropriate folders

## Prerequisites

- Windows OS
- Microsoft Outlook (installed and configured)
- Python 3.8 or higher
- Chrome browser (for form automation)
